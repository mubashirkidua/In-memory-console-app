# Quickstart Guide: In-Memory Python Console Todo Application

## Prerequisites
- Python 3.13 or higher
- No additional dependencies required

## Setup
1. Ensure Python 3.13+ is installed on your system
2. Clone or download the project files
3. Navigate to the project directory

## Running the Application
1. Open a terminal/command prompt
2. Navigate to the project directory
3. Run the application:
   ```bash
   python phase-1/src/main.py
   ```
4. The application menu will be displayed

## Using the Application
1. The main menu will display available options:
   - Add a new todo
   - View all todos
   - Update an existing todo
   - Delete a todo
   - Mark a todo as complete/incomplete
   - Exit the application

2. Follow the on-screen prompts to perform operations
3. Enter the corresponding number for the operation you want to perform
4. Follow the specific prompts for each operation

## Example Workflow
1. Select "Add a new todo" to create a task
2. Enter the title and description when prompted
3. Select "View all todos" to see your tasks
4. Select "Mark a todo as complete/incomplete" to update task status
5. Continue using the application as needed
6. Select "Exit the application" when finished

## Notes
- All data is stored in memory only and will be lost when the application exits
- The application supports up to 1000 todos simultaneously
- All text input supports standard ASCII characters
